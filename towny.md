# 🏛️ Полный гайд по командам плагина Towny

Этот справочник поможет вам быстро найти нужную команду для управления городом, нацией или своим участком. Для быстрого поиска нажимайте `Ctrl + F` или используйте строку поиска в меню слева.

---

## 👤 Основные команды игрока
Эти команды доступны каждому жителю сервера.


| Команда | Описание |
| :--- | :--- |
| `/resident` (или `/res`) | Посмотреть свою статистику (баланс, город, участки) |
| `/res [ник]` | Посмотреть информацию о другом игроке |
| `/res friend add [ник]` | Добавить игрока в список друзей |
| `/res friend remove [ник]` | Удалить игрока из списка друзей |

> 💡 **Совет:** Друзья из списка `/res friend` могут получить доступ к вашим личным сундукам или постройкам, если вы настроите соответствующие права на своих участках.

## 🏡 Управление городом (Для Мэров и Замов)
Команды для создания города, расширения территорий и управления жителями.

### 🔹 Первые шаги и экономика

| Команда | Описание |
| :--- | :--- |
| `/town new [Название]` | Создать новый город (снимает деньги со счета) |
| `/town` (или `/t`) | Открыть информационное окно вашего города |
| `/t deposit [сумма]` | Положить деньги в банк города (налог снимается отсюда!) |
| `/t withdraw [сумма]` | Снять деньги из банка города (доступно только Мэру) |

### 🗺️ Территории и расширение

| Команда | Описание |
| :--- | :--- |
| `/t claim` | Присоединить чанк, в котором вы стоите, к городу |
| `/t unclaim` | Отказаться от чанка, в котором вы стоите |
| `/t claim outpost` | Присоединить удаленный чанк (аванпост) отдельно от города |

### 👥 Жители и должности

| Команда | Описание |
| :--- | :--- |
| `/t add [ник]` | Пригласить игрока стать жителем вашего города |
| `/t kick [ник]` | Выгнать игрока из города |
| `/t rank add [ник] [ранг]` | Назначить жителю должность (например, `assistant` — зам) |

## 🗺️ Команды управления участками (Plots)
Продажа земли жителям внутри города и настройка прав.


| Команда | Описание |
| :--- | :--- |
| `/plot forsale [цена]` | Выставить чанк, на котором стоите, на продажу |
| `/plot claim` | Покупка участка жителем (игрок должен стоять на продаваемом чанке) |
| `/plot toggle pvp` | Включить или выключить ПВП на конкретном участке |
| `/plot set perm [resident/friend/ally/outsider] [on/off]` | Настройка детальных прав доступа к чанку |

---

## 👑 Команды наций (Военно-политический блок)
Команды для создания государств, объявления войн и заключения союзов.


| Команда | Описание |
| :--- | :--- |
| `/nation new [Название]` | Создать нацию (доступно только Мэрам городов) |
| `/nation add [Название_Города]` | Пригласить другой город вступить в вашу нацию |
| `/nation ally add [Название_Нации]` | Заключить официальный союз с другой нацией |
| `/nation enemy add [Название_Нации]` | Объявить другую нацию врагом (для начала войны) |
